# SUDOTEK Local Version (WebLLM)

Fully offline version that runs entirely on your laptop using WebLLM.

## How to Run

1. Open this folder in VS Code
2. Install the "Live Server" extension
3. Right-click `index.html` → "Open with Live Server"
4. Load a model from the top bar (first time takes 30-90 seconds)
5. Enjoy fully local inference!

## Models Available
- Llama-3.2-1B (fast)
- Phi-3-mini (smart)
- SmolLM-135M (tiny)

No internet needed after the model downloads once.
