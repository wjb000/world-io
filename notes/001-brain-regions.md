# Brain Regions Architecture

## Overview
The system is divided into specialized brain-like regions, each containing clusters of neural nodes.

### Proposed Regions:
- **Prefrontal Cortex** — High-level planning, goal decomposition, reasoning
- **Hippocampus** — Episodic memory, context retrieval, long-term storage
- **Amygdala** — Emotional weighting, priority/urgency scoring
- **Visual/Spatial Cortex** — Image understanding, spatial relationships
- **Language Cortex** — Linguistic processing, nuance detection
- **Motor/Executive** — Action planning, tool use, output formatting

Each region will have its own set of specialized mini-LLMs.
